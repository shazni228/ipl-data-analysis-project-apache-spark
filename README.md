# 🏏 IPL Data Analysis Project Using Apache Spark

## 📌 Overview

This project focuses on analyzing **Indian Premier League (IPL)** cricket data using **Apache Spark** and **PySpark**. The goal is to perform large-scale data processing, explore match and player statistics, and extract meaningful insights from IPL datasets using distributed data processing techniques.

The project demonstrates practical experience with **Big Data analytics, Spark DataFrames, SQL operations, data transformation, and exploratory data analysis**.

Run the cells to execute the Spark analysis.
